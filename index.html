Instalasi LAN Menggunakan MikroTik

Materi Jaringan Komputer - TKJ

1. Pengertian LAN

LAN (Local Area Network) adalah jaringan komputer dengan area terbatas seperti sekolah, kantor, atau rumah.

2. Alat dan Bahan

  • Router MikroTik
  • Switch
  • Kabel UTP
  • PC / Laptop
  • Winbox

3. Topologi Jaringan

Menggunakan topologi star dengan MikroTik sebagai pusat jaringan (gateway).

4. Konfigurasi IP Address

IP Router : 192.168.10.1/24
Interface : ether2

5. DHCP Client

Aktifkan DHCP Client pada ether1 untuk mendapatkan IP dari ISP.

6. NAT

Tambahkan NAT masquerade agar jaringan LAN bisa terhubung ke internet.

7. Pengujian

  • Ping gateway
  • Ping antar PC
  • Ping 8.8.8.8

8. Kesimpulan

Konfigurasi LAN MikroTik yang benar akan menghasilkan jaringan yang stabil dan terkoneksi internet.
