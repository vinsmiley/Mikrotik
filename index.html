Instalasi LAN Menggunakan MikroTik

By Kevin Saputra

Find Me On My Social Media

1. Pengertian LAN

LAN (Local Area Network) adalah jaringan komputer dengan area terbatas seperti sekolah, kantor, atau rumah.

2. Alat dan Bahan

  • 1 Buah Router MikroTik
  • 2 Buah Switch
  • 5 Buah Kabel UTP
  • 3 Buah PC / Laptop
  • Software Winbox

3. Setting Interface

  • Klik System > Identity > Isi Sesuai Nama Kalian > Apply > Oke

  • Klik Interfaces > ether1 ubah jadi (ether1-internet) > Apply > Oke, ether2 ubah jadi (ether2-switchA) > Apply > Oke, ether3 ubah jadi (ether3-switchB) > Apply > Oke

    • 4. Setting DNS

    Klik IP > DNS

    Di Kolom Servers Isi 8.8.8.8 , Lalu Klik Panah Kebawah dan Isi 8.8.4.4

    5. Konfigurasi IP Address

    Klik IP > Addresses

    IP Router : 192.168.(Nomor Absen Kalian).1/24
    Interface : ether2-switchA Klik Apply > Oke
    IP Router : 192.168.(Nomor Absen Kalian + 10).1/24
    Interface : ether3-switchB Klik Apply > Oke

    6. DHCP Client

    Klik IP > DHCP Client > DHCP

    Interface Pilih ether1-internet, Klik Apply > Oke.

    7. NAT

    Klik IP > Firewall > NAT > Klik Tanda +

    Di Tab General, Kolom Chain Pilih srcnat, Kolom Out. Interface Pilih ether1-internet

    Di Tab Action, Kolom Action Pilih Masquerade

    Klik Apply > Oke

    8. Setting Routes

    Klik IP > Route > Klik Tanda +

    Di Tab General, Kolom Dst. Address Biarkan 0.0.0.0/0, Kolom Gateway Isi IP Gateway Yang Ada Pada ether1-internet

    Klik Apply > Oke

    9. Pengujian

    • Ping gateway
    • Ping antar PC
    • Ping 8.8.8.8

    10. Kesimpulan

    Konfigurasi LAN MikroTik yang benar akan menghasilkan jaringan yang stabil dan terkoneksi internet.
